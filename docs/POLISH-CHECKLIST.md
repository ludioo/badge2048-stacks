# UI/UX Polish Checklist

Use this checklist to validate UI/UX quality before demo.

## Visual and Layout

- [x] Spacing is consistent across sections
- [x] Typography hierarchy is clear (heading, subheading, body)
- [x] Color contrast is sufficient for text and buttons
- [x] Padding is consistent on cards and sections

## Interaction States

- [x] Hover/focus states are consistent for all buttons/links
- [x] Disabled/loading states are clear and unambiguous
- [x] Transitions feel smooth and not overdone

## Game Feel

- [x] Slide/merge/spawn animations are smooth
- [x] Shake/pulse feedback is not too aggressive
- [x] Sound/haptics toggles work and are clear

## Badge Experience

- [x] Visual distinction for owned/claimable/locked is clear
- [x] Claim flow feels fast and confirmed
- [x] Empty state is informative and actionable

## Accessibility

- [x] Keyboard navigation works for menus and dialogs
- [x] Focus rings are clearly visible
- [x] aria-live covers key feedback

## Content Quality

- [x] All user-facing copy is in English
- [x] No typos in headings or descriptions
